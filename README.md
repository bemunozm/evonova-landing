# EVONOVA Landing Page

Landing page oficial de EVONOVA construida con tecnología de vanguardia para ofrecer una experiencia de usuario de alto rendimiento.

## 🚀 Tecnologías

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Componentes:** [shadcn/ui](https://ui.shadcn.com/)
- **Lenguaje:** TypeScript
- **Calidad de código:** ESLint + Prettier

## 📋 Estructura del Proyecto

```text
src/
├── app/            # App Router (páginas y rutas)
├── components/     # Componentes de UI (shadcn/ui y propios)
├── lib/            # Utilidades y configuración
├── styles/         # Estilos globales
└── types/          # Definiciones de tipos TypeScript
```

## 🛠️ Instalación

1. Clonar el repositorio:
   ```bash
   git clone <url-del-repo>
   cd evonova-landing
   ```

2. Instalar dependencias:
   ```bash
   npm install
   ```

3. Ejecutar entorno de desarrollo:
   ```bash
   npm run dev
   ```

## ⚙️ Scripts Disponibles

- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Genera la versión de producción.
- `npm run lint`: Ejecuta el analizador de código.
- `npm run format`: Formatea el código usando Prettier.

## 🏗️ Calidad de Código

Este proyecto utiliza **ESLint** y **Prettier** para mantener un código limpio y consistente.
- Ejecuta `npm run lint` antes de realizar un commit.
- El formato se aplica automáticamente mediante Prettier configurado.
