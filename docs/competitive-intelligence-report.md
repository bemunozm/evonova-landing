## 3. Referentes Mundiales de Diseño (Vercel / Linear / Stripe)

| Referente | Características Clave |
|---|---|
| **Vercel** | Fondo negro, tipografía blanca gigante, copy ultra-directo, cero ruido |
| **Linear** | Dark mode, gradientes aurora boreal, columna única, espaciado radical |
| **Stripe** | Gradientes orgánicos animados en hero, bloques densos bien jerarquizados, copy centrado en el cliente |

### Tendencias 2025/2026 para Agencias B2B Tech
- Interfaces "calmas" y minimalistas — menos es más
- Narrativa humana: casos de estudio con estilo editorial/periodístico
- Chat-first: CTA conversacional (WhatsApp/Chatbot) por encima del formulario
- Pricing transparente como diferenciador de confianza
- Dark mode como opción estándar para audiencia tech

---

## 4. Design Brief para Rediseño EVONOVA

**Estética objetivo:** "Linear meets Stripe" — dark mode con gradientes aurora (negro, índigo profundo, verde-teal como acento), tipografía serif/editorial para headlines.

### Estructura de la Landing Page
1. **Nav** — logo + links mínimos + CTA "Agendar"
2. **Hero** — Headline impactante + subheadline + CTA primario + trust bar con métricas reales
3. **Casos de Éxito** — 3 cards premium (Pira.cl, OTEC WP+Moodle, LangGraph)
4. **Servicios** — 3 pilares con iconografía (Automatización, Software, Consultoría)
5. **Precios Orientativos** — 3 paquetes (Starter / Growth / Enterprise) ← diferenciador único
6. **CTA Final** — "Agenda tu diagnóstico gratuito de 15 min"
7. **WhatsApp flotante**
